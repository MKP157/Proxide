Project title: Peroxide

Project Members
  • Josh Hickman
  • Aiden Manuel
  • Matthew Peterson
The goal of our project has been to get a basic Rust web-server working.

This allows us to explore the basics of responding to web requests, working in Rust and using its threading capabilities.

Primary goal is to be able to provde HTML pages in response to normal web navigation of our site. With optional objective of providing videos encoded by our Architecture project.
